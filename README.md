## System Dependencies

`sdl2` together with its dependencies are required to be installed on your hostmachine.
On arch-based distros you can use
`sudo pacman -Syu sdl2 sdl2_gfx sdl2_image sdl2_mixer sdl2_ttf`
to install them.
